# Analisis Exploratorio - Ventas Superstore
Análisis exploratorio de datos de ventas usando Python.

Este proyecto presenta un análisis exploratorio de datos (EDA) realizado sobre un conjunto de datos de ventas comerciales, con el objetivo de identificar patrones, tendencias y relaciones relevantes entre ventas, descuentos y ganancias que apoyen la toma de decisiones.

## Objetivos:
- Analizar el comportamiento de las ventas.
- Evaluar la relación entre descuentos y rentabilidad.
- Identificar categorías y productos con mayor impacto en las ganancias.
- Explorar la evolución de las ventas en el tiempo.
- Identificar las regiones con mayor número de ventas y ganancias. 

## Herramientas utilizadas
- Python
- Pandas
- Matplotlib
- Google Colab

##Conclusiones

- La categoría **Thecnology** presenta mayor rentabilidad.
- Se identifica una relación inversa entre el porcentaje de descuentos y el valor en las ganancias.
- Las regiones con mayores ventas representan también las zonas con mayores ganancias.
- Al pasar de los meses las ventas comerciales tiende a tener un comportamiento creciente entre los meses de Noviembre. 
- Algunos productos estás generando pérdidas, por lo cual se recomienda tomar medidas sobre ellos para no tener afectaciones en la rentabilidad.
